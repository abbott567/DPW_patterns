Draft Third party details

Standard pattern

Use this when:

  • the user needs to declare that they are completing the transaction on behalf of someone else.

Research notes

User research for Carer's Allowance found that asking people early in the transaction if they were completing it on behalf of the carer prevented them from entering their own details.

Are you applying for Carer's Allowance for yourself?

Are you applying for Carer's Allowance for yourself?

Fill in the rest of the form as if you're the carer. For example, if asked for 'your address' put the address of the person doing the caring.

View code snippet
HTML JS 
                  
    <h1 class="heading-xlarge">Are you applying for Carer's Allowance for yourself?</h1>
    
    <form>
      <fieldset>
        <div class="form-group">
          <label class="block-label" for="carer-yes">
            <input id="carer-yes" type="radio" name="carer" value="yes">
            Yes, you're the Carer
          </label>

          <label class="block-label" for="carer-no">
            <input id="carer-no" type="radio" name="carer" value="no">
           No, you're applying on behalf of the carer
          </label>
        </div>
      </fieldset>

        <div class="js-hidden" id="third-party-details">
          <div class="form-group">
            <fieldset>
              <label class="form-label-bold" for="third-party-name">
               Your name and organisation
             </label>
             <input type="text" class="form-control" id="third-party-name" name="third-party-name">
           </fieldset>
         </div>
         <p>Fill in the rest of the form as if you're the carer. For example, if asked for 'your address' put the address of the person doing the caring.
       </div>
    </form>
                  
                
                  
    $(document).on('click', 'input[type="radio"]', function (e) {
      if ($(e.target).attr('id') === 'carer-yes' || $(e.target).attr('id') === 'carer-no') {
        $('#third-party-details').show();
        if ($(this).attr('id') === 'carer-no') {
          $('#third-party-details').show();
        } else {
          $('#third-party-details').hide();
        }
      }
    });
