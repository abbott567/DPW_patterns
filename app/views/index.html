DWP design patterns

This guide shows you patterns for DWP services. You should use it when the GOV.UK pattern library doesn't have a solution to your design problem.

Visual style

If you're working on a citizen-facing service that's hosted on GOV.UK, use the GOV.UK elements guide (which includes code snippets).

If you're working on a service that isn't hosted on GOV.UK or a service for DWP staff, use the DWP elements guide.

Form elements

  • Add item(s) to a list
  • Check your answers
  • Entering bank details
  • Document upload
  • Save and exit
  • Declarations

Staff interfaces

  • Admin menus
  • Search and filter
  • Task lists and completion indicators
