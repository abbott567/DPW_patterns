Add items to a list

Use this pattern when you're asking the user to enter a list of things (eg medical conditions or names of people). It allows the user to add as many fields as they need, one at a time, so they aren't put off by a page of empty fields.

Standard version - horizontal fields

This is how the first list item appears. Note the grey colour of the button below.

When the user clicks the 'Add another' button, a second set of text entry fields appears below, separated by a keyline. The 'Add another' button moves below the new fields ready for the user to click it again.

A 'Remove' link also appears next to both of the items, which allows the user to quickly delete an entry if they need to.

If there are only 2 items in the list, the 'Remove' button will disappear when it's clicked the first time (because the user shouldn't be able to remove all items).

Variant version - vertical fields

Use this version when you're asking for more fields than will fit on the page horizontally.

Demo version

See how it works with this demo version. The number of items you can add is limited to 5 for the purposes of the demo.
