Admin menus

Standard pattern

Use this when:

  • the user needs access to several admin areas (the menu is left-aligned and displays the title of each admin area)
  • the user needs access to generic action links, like ‘Log out’ (these are displayed in the grey bar below the top black bar, with new links added to the right)
