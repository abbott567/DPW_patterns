Admin menus

Standard pattern

Use this pattern when creating interfaces that will be used only by internal departmental staff and third-party suppliers. The pattern is made up of two sections; the main left-hand navigation and the page sub-navigation.

Main navigation

The main admin menu navigation is left-aligned and displays the titles of each admin area. When the user selects one of these the area's content is displayed in the central panel.

There is also a strip beneath the top black bar which is where any action links can be placed eg a 'log out' action. Any actions placed in here are right-aligned and new ones are added to the right.

          $(this).parents('li').addClass('active');

      });

Secondary navigation

It may be necessary to have a secondary level of navigation within an area and this pattern is for that purpose. Selecting a title changes the content below the keyline and the coloured focus mark moves underneath the title.
